  * &#x2705; 统一接口
  * &#x2705; 特征工程：提取有效特征（针对某一形态的特征进行提取；然后就可以使用多个形态模型进行叠加预测），将模型的准确性直接和信号过滤器的合理性挂钩，只需要不断实现信号过滤器的接口就可以不断地优化训练集的质量
    * 特征数组的归一化/标准化支持
    * &#x2705; 分类平衡器：保证输出训练集的分类数量平衡（为降低复杂度，只提供二分分类，可以多模型叠加进行分析预测）
      * 优先去除争议信号：比如被重置了的信号，代表所关注形态上的争议性，成为噪音的可能性较大；其他形态可以使用其他模型进行叠加预测
      * 然后去除弱信号：比如对未来影响小的信号（代表了一种模糊性，成为噪音的可能性较大）；模糊性往往代表交易区间，可以使用与交易区间有关的模型进行叠加预测
    * &#x2705; 信号重置器（过滤器）：只关注特定形态进行训练
      * 顺势形态（涨1跌-1）：重置回踩和反转信号为0
      * 以下形态定义可能不完善，只是提供一种思路
        * 反转形态（低概率，反涨1反跌-1）：重置回踩和顺势信号为0
        * 趋势回踩形态（高概率，回踩-1反弹1）：重置反转和顺势信号为0
        * 交易区间形态
        * 趋势性交易区间形态
    * 接口有点乱，需要重构
